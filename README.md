## todogo
a todo-list tool by typescript wechat miniprogram 

## 1. 项目概述

**Todogo** 是一款基于微信小程序的待办事项管理与习惯养成应用。通过植物生长 gamification（游戏化）机制，激励用户完成日常任务，培养良好习惯。

---

## 2. 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | 微信小程序 (WeChat Mini Program) |
| 开发语言 | TypeScript |
| UI 框架 | 原生小程序组件 |
| 组件库 | glass-easel |
| 构建工具 | 微信开发者工具 + TypeScript 编译器 |
| 样式 | WXSS |

---

## 3. 核心功能

### 3.1 待办事项管理
- 创建任务（标题、描述、优先级）
- 标记任务完成/未完成
- 编辑和删除任务
- 三种优先级：普通、重要、紧急
- 按状态和优先级排序

### 3.2 日历视图
- 月历展示
- 每日任务数量标识
- 月份切换导航
- 选择日期查看/添加任务
- 快捷返回今日

### 3.3 备忘功能
- 创建文字备忘
- 支持添加图片（最多9张）
- 删除备忘
- 显示创建时间

### 3.4 成长系统（游戏化）
通过完成任务推动植物生长：

| 等级 | 名称 | 图标 | 完成任务数 |
|------|------|------|------------|
| 0 | 小种子 | 🌰 | 0-2 |
| 1 | 发芽 | 🌱 | 3-7 |
| 2 | 幼苗 | 🌿 | 8-15 |
| 3 | 小树 | 🌳 | 16-29 |
| 4 | 大树 | 🌲 | 30-49 |
| 5 | 果树 | 🍎 | 50+ |

### 3.5 数据统计
- 今日任务总数
- 已完成任务数
- 待办任务数
- 距离下一等级进度

---

## 4. 总结

Todogo 是一款功能完整的待办事项管理小程序，具备：

- ✅ 待办事项完整 CRUD 操作（含优先级）
- ✅ 日历视图集成
- ✅ 文字/图片备忘功能
- ✅ 植物生长游戏化系统
- ✅ 本地数据持久化
- ✅ 无需后端即可独立运行
